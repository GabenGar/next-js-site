# Gabengars

## Installation
Clone repo:
```sh
git clone https://github.com/GabenGar/next-js-site.git
cd next-js-site
npm install
```

Create .env file and set the values:
```
npm run env
```

## Develop

```sh
npm run dev
```

## Build

```sh
npm run build
```

The result will be inside `.next` folder.
